Documento gerado para teste de conversão HTML para PDF

Introdução

Este é um parágrafo de exemplo que serve para demonstrar a renderização de texto simples. O objetivo deste documento é fornecer um conteúdo variado para que o conversor de HTML para PDF possa ser testado em diferentes cenários, incluindo formatação de texto, imagens e listas.

Recursos Principais

A conversão deve ser capaz de lidar com os seguintes elementos:

  • Títulos de diferentes níveis (h1, h2).
  • Parágrafos com texto contínuo.
  • Listas não ordenadas como esta.
  • Estilos CSS incorporados (cores, fontes, espaçamento).
  • Imagens.
